# 学术资源集合

这是一个集合了多个学术研究资源的网页应用，UI设计采用黑白极简风格。

## 包含的资源

1. **Elicit** - AI驱动的研究助手
2. **Research Rabbit** - 文献发现与整理工具
3. **厦门大学图书馆** - 高校学术资源门户
4. **Kimi** - AI对话与知识助手

## 如何使用

1. 直接打开 `index.html` 文件在浏览器中查看
2. 点击任意卡片将跳转到对应的网站

## 图片资源

请将以下图片文件放置在 `images` 文件夹中：
- `xmu_emblem.png` - 厦门大学校徽
- `xmu_name.png` - 厦门大学校名

## 技术特点

- 简洁现代的黑白风格UI设计
- Notion风格的图标系统
- 响应式布局，适配不同屏幕尺寸
- 使用CSS变量实现主题色彩管理
- 精美的卡片悬停动画效果
- 使用Feather图标库

## 适用场景

本应用为"科研工作坊：AI时代口译研究设计与方法"设计，集合了常用的学术研究工具，方便工作坊参与者快速访问相关资源。
